
##  页面com-go-top组件说明

|  属性				|	说明						|	类型			|	默认值				|
|-------------------|-------------------------------|-------------------|-----------------------|
right				|	距离页面右边距离				|	String Number	|	30（rpx）			|
bottom				|	距离页面底部距离				|	String Number	|	30（rpx）			|



## 示例
```html
用法：
<com-go-top ref="goTop" right="30" bottom="30" />
```


```js
this.$refs['goTop'].show();
```
