
##  页面com-show-price组件说明

|  属性				|	说明								|	类型				|	默认值			|
|-------------------|-----------------------------------|-------------------|-------------------|
price				|	金额								|	String			|	middle			|
color				|	颜色								|	String			|	#E54D42			|
bold				|	是否粗体							|	NULL			|	false			|
minSize				|	小字体大小 rpx					|	String Number	|	取消				|
maxSize				|	大字体大小 rpx					|	String Number	|	确认				|


## 示例
```html
用法：
<com-show-price price="56.00" minSize="24" maxSize="36" bold />
```
